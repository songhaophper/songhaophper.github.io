getLocalData

获取webview本地存储数据

1.WVJsBridge.getLocalData(OPTION,CALLBACK)

名称 类型 必填 描述
arr array 是 需要获数据的key
callback function 是 回调函数拿到数据

2. 代码示例

ES6箭头函数形式

WVJsBridge.getLocalData(['test1','test2'],(data)=>{
      const testData=JSON.parse(data)
      const test1 = testData.test1
      const test2 = testData.test12
})

ES5 普通回调函数形式

WVJsBridge.getLocalData(['test1','test2'],function(data){
      var testData=JSON.parse(data)
      var test1 = testData.test1
      var test2 = testData.test12
})
