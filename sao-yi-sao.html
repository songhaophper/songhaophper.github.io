设备功能

扫一扫

1.WVJsBridge.scan(CALLBACK)

名称 类型 必填 描述
callback function 是 回调函数拿到数据

2. 代码示例

ES6箭头函数形式

WVJsBridge.scan((data)=>{
    // data 为base编码字符串
    // 进行解码并转json对象
    const orderData = JSON.parse(data)
    /*拿到解密之后的base64 code值 (sdk并没有帮你解码，请自行解码) */
    const scanCode = Base64.decode(orderData.scanCode)
    const codeJSON = JSON.parse(scanCode)
    const codeData = codeJSON.code
})

ES5 普通回调函数形式

WVJsBridge.scan(function(data){
    // data 为base编码字符串
    // 进行解码并转json对象
    var orderData = JSON.parse(data)
    /*拿到解密之后的base64 code值 (sdk并没有帮你解码，请自行解码) */
    var scanCode = Base64.decode(orderData.scanCode)
    var codeJSON = JSON.parse(scanCode)
    var codeData = codeJSON.code
})
